# Safari Web Extension resources

This directory contains the source resources used by Bili Focus:

- `manifest.json` — extension manifest and Bilibili permissions
- `preload.css` — prevents the original homepage from flashing
- `redirect.js` — replaces the homepage without changing the address bar
- `home.html`, `home.css`, `home.js` — focus homepage UI and interactions
- `icons/` — extension icons

For installation and signing instructions, see the [project README](../README.md).

After editing these files, copy the updated resources to:

```text
BiliFocusXcode/Bili Focus/Bili Focus Extension/Resources/
```
